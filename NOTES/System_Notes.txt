Системка С++

Lvl 1
===================================
Кодировка - определенный набор символов, которым к каждому символу сопоставляеться последовательность длинной один или несколько байтов

В кодировке ANSI каждому символу выделяется один байт (8 бит), но существуют языки для которых одного байта не достаточно

Для поддержки таких языков создали UNICODE, в котором каждый символ - 2 байта (Диапазон: 0 - 65536)

Используя UNICODE - разработчик может в любой момент подключить локализацию

Тип wchar_t - тип данных для UNICODE символов

Для преобразования из ANSI в UNICODE - используется префикс L (eg. wchar_t roflan[15] = L"Yo";)

Макросы для работы с UNICODE - #include <tchar.h>

На этапе создания проекта програмист абстрагирован от типа кодировки, т.е точно не знает какую кодировку лучше использовать
Таким образом лучше создать универсальный код, который легко подстроиться под любую кодировку, для этого используються макросы Windows

wcout - выводит UNICODE строку

Для явного подключения кодировки UNICODE - используется символическая константа UNICODE - #define _UNICODE

Lvl 6
=========================================
Функция SendMessage - отправляет ассинхронное сообщение в оконную процедуру. Оконная процедура немедленно обрабатывает сообщение
PostMessage - помещает сообщение в очередь сообщений приложения, обработка происходит через цикл обработки сообщений
GetMessage - извлекает сообщение и передает оконной процедуре
PostMessage - используеться для отправки синхронных сообщений
